# Articles App — Frontend

Фронтенд‑часть приложения для работы со статьями, версиями, вложениями и аутентификацией.  
Использует React, React Router и Axios.

## Технологии
- React (CRA)
- React Router v6
- Axios
- React Quill (редактор текста)
- WebSocket клиент
- JWT аутентификация

---

# Установка и запуск

```bash
cd frontend
npm install
npm start
